# `yc-bench company-status`

Get a comprehensive overview of your company's current financial status, reputation, active tasks, and employee payroll obligations.

## Usage

```bash
yc-bench company-status
```

## Parameters

This command takes no parameters.

## Response

Returns current company metrics:

<ResponseField name="funds" type="number">
  Current available cash in dollars
</ResponseField>

<ResponseField name="prestige" type="number">
  Current prestige score (reputation)
</ResponseField>

<ResponseField name="employeeCount" type="number">
  Total number of employees
</ResponseField>

<ResponseField name="monthlyPayroll" type="number">
  Total monthly salary obligations in dollars
</ResponseField>

<ResponseField name="activeTasks" type="number">
  Number of tasks currently in progress
</ResponseField>

<ResponseField name="plannedTasks" type="number">
  Number of accepted tasks not yet started
</ResponseField>

<ResponseField name="runwayMonths" type="number">
  Months of runway at current burn rate
</ResponseField>

<ResponseField name="nextPayrollDate" type="string">
  Date of next payroll payment (ISO 8601 format)
</ResponseField>

## Example

```bash
# Check company status
yc-bench company-status
```
